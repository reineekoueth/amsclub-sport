## 🚀 Fonctionnalités

### 👤 Utilisateur

* Inscription / Connexion sécurisée (JWT)
* Gestion du profil
* Inscription aux activités
* Désinscription

| Vue.js         | Node.js    | MySQL           |
| Axios          | Express.js |                 |
| Vue Router     | JWT        |                 |
